Root your unit


On your head unit

  1. If you do not have it already, you should download my apk from the Releases page on to your head unit.
    When you start the apk you will see the following screen:


    Your unit is not fully rooted. You already knew this otherwise you would not try to root it. Simply press OK.

  2. Next you will see:

    On Android 6.0.1 adb over wifi is by default disabled. This will be displayed by the button "Enable adb over WiFi". To root your unit you need to have adb over Wifi enabled. If you tap the button, it will be enabled.
    If the button mentions "Disable adbover Wifi", you don't have to do anything.
  3. Now you need to know the ip-address of your head unit. Tap the top-left menu button   and select the option "My WiFi address".
    You need it for the script on your laptop/pc.
    You will see something like this:

On your pc:


Linux: First install adb via your package manager (like on Ubuntua/Debian like distributions: "sudo apt-get install adb")
  1. Download the latest SuperSu version from the Releases page onto your pc/laptop. It will be called something like SuperSU V x.yz.
  2. Unzip it somewhere on your pc/laptop and keep the internal folder structure instact, so not all files in one folder but with sub-folders.
    • On Windows: Run CMD.exe as Administrator.
    • On Linux like OSes: Open a terminal.
  3. Change to the folder where you unzipped the files and change to the folder where the copy_install.sh and copy_install.bat are located.
    • On Windows: Run the copy_install.bat script with the IP address of your Head Unit as a parameter: copy_install.bat 192.168.178.50 (for example)
    • On linux: Run the copy_install.sh script with the IP address of your Head Unit as a parameter:./copy_install.sh 192.168.178.50 (for example)
  4. Wait until the script finishes.
  5. Reboot your Head Unit.
You should now have the SuperSU application working with modified SELinux policy settings. This will allow you to give other apps root access as well.
