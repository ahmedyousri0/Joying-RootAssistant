This site contains:

  • A script to root your unit
  • An apk to install a number of additional or modded apks

The rooting script and the necessary files to root your Sofia 3GR Intel Joying unit can be downloaded from the Releases page. It doesn't matter which Intel Joying unit, which resolution or whether you are on a 5.1.1 or a 6.0.1 Android release version.
It doesn't even matter whether it is a Joying as long as it is a Sofia 3GR unit.
See this page for a HowTo.


The Joying Extra Tools apk can also be downloaded from that Releases page.


The Joying Extra Tools apk allows you to:
  • Enable or disable adb over WiFi
  • Upgrade the binaries of your already rooted unit
  • Update busybox (see jrummyapps)
  • Install rsync
  • Install Viper4Android
  • Install several Sofia Server mods
  • Install Joying Settings.apk or Standard Android Settings.apk
  • Install Radio mods (examples)
  • Install other mods like Music Player, Video Player, Bluetooth (examples)
  • (Continuously) measure temperatures and CPU frequencies of your unit
  • Alter the DPI of your unit
  • Expand the hosts file to prevent ads, banners, hijackers, etc. (see http://winhelp2002.mvps.org/hosts.htm)
  • Reboot and "Hot Reboot" options
  • Option to wipe /data to do a "factory defaults" reset.
  • some more stuff to come..

Note that most of the mods are not from me, but made by others.

The aim is to keep "everything" as updated as possible.

The app is created for Joying Intel Sofia 3GR units. Most functionality will probably run just as well on other brand Sofia 3GR units but for the time that is blocked (The app tests whether it is running on a Joying). Some functionality is for both 5.1.1 and 6.0.1. The apk will detect which version you are running and will offer, when relevant, the correct options for your android version.
